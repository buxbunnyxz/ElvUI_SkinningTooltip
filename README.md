Elvui plugin for wotlk 3.3.5a
- Adds a text to the tooltip IF you have skinning, only on skinnable mobs with a text of the required skinning level.
- Enable or disable via the ElvUI panel.
- Text color from grey to red
  - Grey : no skill gain chance.
  - Green : very low skill gain chance.
  - yellow : moderate skill gain chance.
  - orange : 100% skill gain chance.
  - red : no skill gain chance.
